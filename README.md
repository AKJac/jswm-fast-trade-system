# 极速外贸管理系统

鸿思特科技有限公司开发的综合外贸管理系统，采用Vue 3 + Element Plus构建现代化前端界面。

## 项目概述

极速外贸管理系统是一款专为外贸企业设计的全流程管理软件，涵盖从客户管理、产品管理、报价管理、订单管理、采购管理、出运管理、库存管理到财务管理的完整业务链条。

## 技术栈

- **前端框架**: Vue 3 (Composition API)
- **UI组件库**: Element Plus 2.3.8
- **构建工具**: Vite 4.4.0
- **样式预处理**: Sass/SCSS
- **路由管理**: Vue Router 4
- **图表库**: ECharts 5.4.2
- **HTTP客户端**: Axios 1.4.0
- **日期处理**: Day.js 1.11.9

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```
