## 项目概述

本项目是一个基于 Vue 3 + Element Plus 的极速外贸管理系统，已完成了所有核心模块的增删改查功能。系统采用现代化的前端技术栈，提供完整的业务管理解决方案。

## 技术栈

- **前端框架**: Vue 3 (Composition API)
- **UI组件库**: Element Plus
- **样式处理**: SCSS
- **路由管理**: Vue Router
- **状态管理**: Vue 3 Reactive API
- **图表库**: ECharts (用于数据可视化)

## 已完成的模块功能

### 1. 客户管理模块
- **功能**: 客户信息管理、客户分类、跟进进度跟踪
- **组件**: CustomerList.vue, CustomerForm.vue
- **特点**: 完整的客户生命周期管理

### 2. 产品管理模块
- **功能**: 产品信息管理、产品参数设置、价格管理
- **组件**: ProductList.vue, ProductForm.vue
- **特点**: 产品图片上传、库存关联

### 3. 报价管理模块
- **功能**: 报价单管理、产品明细、价格计算
- **组件**: QuotationList.vue, QuotationForm.vue
- **特点**: 自动价格计算、有效期管理

### 4. 订单管理模块
- **功能**: 订单信息管理、订单状态跟踪、金额计算
- **组件**: OrderList.vue, OrderForm.vue
- **特点**: 完整的订单流程管理

### 5. 库存管理模块
- **功能**: 库存信息管理、库存数量控制、位置管理
- **组件**: InventoryList.vue, InventoryForm.vue
- **特点**: 库存预警、价值计算

### 6. 采购管理模块
- **功能**: 采购单管理、供应商管理、采购状态跟踪
- **组件**: PurchaseList.vue, PurchaseForm.vue
- **特点**: 采购审批流程

### 7. 财务管理模块
- **功能**: 应收款管理、收款记录、财务统计分析
- **组件**: FinanceOverview.vue, ReceivableForm.vue
- **特点**: 数据可视化、财务图表

### 8. 系统管理模块
- **功能**: 用户管理、角色权限、部门管理
- **组件**: UserManagement.vue, UserForm.vue
- **特点**: 完整的权限控制系统

### 9. 任务管理模块
- **功能**: 任务分配、进度跟踪、工时统计
- **组件**: WorkTask.vue, TaskForm.vue
- **特点**: 任务看板、优先级管理

### 10. 物流管理模块
- **功能**: 物流信息管理、货物跟踪、费用计算
- **组件**: ShippingList.vue, ShippingForm.vue
- **特点**: 多运输方式、地址管理

## 部署说明

### 环境要求
- Node.js 16+
- npm 或 yarn 包管理器

### 安装和运行
```bash
# 安装依赖
npm install

# 开发环境运行
npm run dev

# 生产环境构建
npm run build

# 预览构建结果
npm run preview
```
