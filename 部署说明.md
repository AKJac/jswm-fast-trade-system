## 🚀 部署步骤

### 1. 准备文件

项目已经构建完成，生产文件位于 `dist` 目录中，包含：
- `index.html` - 主页面文件
- `assets/` - 静态资源目录（JS、CSS文件）

### 2. 宝塔面板操作

#### 2.1 创建网站
1. 登录宝塔面板
2. 点击"网站" → "添加站点"
3. 填写域名（如：`your-domain.com`）
4. 选择PHP版本为"纯静态"
5. 点击"提交"

#### 2.2 上传文件
1. 在网站列表中找到刚创建的站点
2. 点击"根目录"进入网站目录
3. 删除默认的 `index.html` 文件
4. 上传 `dist` 目录中的所有文件到网站根目录

#### 2.3 配置伪静态（重要）
由于使用了Vue Router的history模式，需要配置伪静态规则：

**Nginx配置**（在宝塔面板 → 网站 → 设置 → 伪静态）：
```nginx
location / {
    try_files $uri $uri/ /index.html;
}
```

**Apache配置**（如果使用Apache）：
```apache
<IfModule mod_rewrite.c>
  RewriteEngine On
  RewriteBase /
  RewriteRule ^index\.html$ - [L]
  RewriteCond %{REQUEST_FILENAME} !-f
  RewriteCond %{REQUEST_FILENAME} !-d
  RewriteRule . /index.html [L]
</IfModule>
```

## 🔧 常见问题

### 问题1：页面刷新404
**解决方案**: 确保已正确配置伪静态规则

### 问题2：静态资源加载失败
**解决方案**: 检查文件上传是否完整，确保 `assets` 目录存在

### 问题3：HTTPS证书问题
**解决方案**: 在宝塔面板中为域名申请SSL证书
